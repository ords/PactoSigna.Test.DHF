# Hazardous Situation: Healthcare Provider Administers Incorrect Treatment

## Description

A healthcare provider, relying on information displayed by the software system, administers treatment that differs from the intended prescription. This situation can arise from multiple hazards including incorrect dose display or confusing user interface.

## Probability Assessment

**Rating: 3 (Possible)**

- System is used by trained healthcare professionals
- Multiple verification steps exist in clinical workflow
- However, time pressure and fatigue can reduce vigilance
- Similar incidents have occurred with other software systems

## Exposure Frequency

- Software used multiple times daily by clinical staff
- Critical treatment decisions made based on displayed information

## Contributing Factors

- High workload environments
- Multiple patients requiring attention
- Trust in software accuracy
